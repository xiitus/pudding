# プロダクト概要

updated_at: 2026-02-24
更新理由: 実装済みの配布導線検証と安全な永続化方針をプロダクト価値に反映

`pudding` は最低限の操作に絞ったターミナル・ペインマルチプレクサです。zellij にインスパイアされた軽量 TUI ツールとして、複雑な設定なしにすぐ使い始められることを最優先にしています。

## コア能力

- **最小操作でのレイアウト制御**: 再帰分割・リサイズ・隣接交換を同一の操作体系で扱う
- **テンプレート駆動の再利用**: レイアウトを保存・復元し、プロジェクトごとに再現可能にする
- **導入経路の一貫性**: Cargo / npm ラッパーのどちらからでも同じ実行体験に収束させる

## ターゲットユースケース

- tmux / zellij を使うほどでもない軽量な分割ターミナル環境が欲しい開発者
- プロジェクトごとのペイン構成をテンプレートとして再利用したいユーザー
- キーバインドのカスタマイズを最小コストで行いたいユーザー

## 価値提案

「必要最小限の機能のみ」を原則とし、設定ファイルはシンプルな JSON 1 ファイルで完結します。設定・テンプレート・状態は XDG 準拠の単一ディレクトリに安全な権限で保存し、配布導線では相対パス拒否や postinstall 失敗伝播を含む自己検証を行うことで、シンプルさと信頼性を両立します。
