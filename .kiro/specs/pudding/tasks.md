# タスク

1. Rust ワークスペースと `pudding` クレートの作成
2. `core` クレートを作成し、レイアウトモデル（`bite`/`spoon`）と操作（分割/リサイズ/交換）を実装
3. `core` にテンプレート/状態の JSON 読み書きと構造検証を実装
4. `editor` クレートを作成し、AAミニチュアエディタ（TUI）を実装
5. `runtime` クレートを作成し、実行ランタイム（PTY + 描画 + 入力）を実装
6. JSON キーバインド対応
7. `editor/runtime -> core` 依存制約を静的解析で検証（逆依存・循環依存を禁止）
8. npm 配布方式の探索（同梱バイナリ方式 vs 薄い CLI 方式）を実施し、採用案を決定
9. npm パッケージラッパーを実装（pnpm/bun 想定）
10. `pnpm`/`bun` のインストール・起動検証を CI に追加
11. README と基本ドキュメントの作成
12. 名前入力と JSON 構造の検証ロジック実装（fail-closed）
13. 設定破損時の起動失敗契約を実装
14. 追加テストを実装（core の境界値・editor 入力系・runtime 統合系）
15. 配布導線テストを実装（npm install / 実行失敗時のエラー伝播）
16. 静的解析ゲート（fmt/clippy/test）を CI 化
